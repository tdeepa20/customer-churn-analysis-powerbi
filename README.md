# Customer Churn Analysis Dashboard (Power BI)

This Power BI dashboard analyzes customer churn to help identify key factors driving customer retention and loss.

## Overview
Key Performance Metrics (KPIs):
- Total Customers: 5,630  
- Churned Customers: 948  
- Active Customers: 4,682  
- Churn Rate:16.8%

These KPIs give a quick snapshot of the customer retention health.

## Insights
1. Churn Rate by City Tier
Tier 3 customers show the highest churn (21.4%), indicating weaker loyalty compared to Tier 1 cities.

2. Churn Rate by Payment Mode 
Customers using Cash on Delivery (24.9%) and E-Wallet (22.8%) churn more often than those using Debit or Credit Cards.

3. Churn Rate by Login Device
Churn is higher among Computer users (44.06%) compared to Phone users.

4. Active Customers by Gender  
Male customers show higher activity levels (2.8K vs. 1.9K for female).
